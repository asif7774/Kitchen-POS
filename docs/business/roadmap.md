# Product Roadmap

This roadmap reflects planned improvements and new features in priority order. Dates are targets, not guarantees — actual release timing depends on development progress and user feedback.

---

## v1.1 — Kitchen Display Screen Improvements
**Target:** Q3 2026

The Kitchen Display Screen (KDS) exists in v1.0 but requires manual setup. This release makes it seamless.

- Automatic KDS detection on second monitor — no configuration required
- Kitchen staff see pending orders on a dedicated screen without paper slips
- Orders appear automatically when KOT is sent from the billing terminal
- Completed orders can be marked done from the KDS screen
- Visual countdown timer per order to track preparation time

---

## v1.2 — Digital Bill Delivery
**Target:** Q3 2026

Send bills directly to customers' phones — no printer required for the customer copy.

- **UPI QR code on printed receipts** — customers scan to pay, reference auto-captured
- **WhatsApp bill delivery** — send the GST invoice as a PDF directly to the customer's WhatsApp number
- Customer phone number optional at order time; can be added at billing

---

## v1.3 — Multi-Terminal Sync
**Target:** Q4 2026

Allow two or more billing terminals to share the same data over a local network.

- Two or more computers on the same Wi-Fi see the same tables, orders, and menu in real time
- Orders placed on one terminal are immediately visible on all others
- No internet required — works entirely on the local network
- Supports one main terminal and up to three secondary terminals

---

## v1.4 — Cloud Backup
**Target:** Q1 2027

Protect data without relying on manual USB backup routines.

- Automatic daily backup to Google Drive (requires a Google account)
- Restore from cloud backup on any machine
- Backup history with point-in-time restore (last 30 days)
- Manual backup still available alongside cloud backup

---

## v1.5 — Customer Loyalty Programme
**Target:** Q1 2027

Reward repeat customers and encourage return visits.

- Loyalty points earned per visit based on spend amount
- Membership tiers (e.g. Silver, Gold) with different discount rates
- Points redeemable at billing — cashier sees available points and applies discount
- Customer loyalty history in the customer profile screen

## How to Request a Feature

If you need a feature not listed here, raise it with your account contact or open a request at the support channel. Features requested by multiple active users are prioritised.
